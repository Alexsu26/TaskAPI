## Task

T011: Implement user login with bcrypt password verification.

## Work Completed

- Reviewed the learner implementation of `POST /users/login`.
- Confirmed login is wired through `main -> router -> handler -> service -> repository`.
- Confirmed repository lookup by email uses `QueryRow` and maps `sql.ErrNoRows` to `repository.ErrUserNotFound`.
- First review found that login re-hashed the submitted password and compared hash strings, causing correct passwords to fail because bcrypt hashes include salt.
- First review found that invalid credentials were mapped to `400 missing parameter`.
- First review found a handler error branch without `return` after `ctx.JSON`.
- Learner fixed the implementation to use `bcrypt.CompareHashAndPassword`.
- Learner fixed invalid credentials to return `401 Unauthorized` with the same message for wrong email and wrong password.
- Learner fixed the missing `return`.
- Final review accepted T011 and updated task, progress, skill, reflection, review, and session records.
